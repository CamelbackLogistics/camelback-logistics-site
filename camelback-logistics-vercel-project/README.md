# Camelback Logistics Website

Vercel-ready React/Vite project.

## Deploy on Vercel

1. Unzip this folder.
2. Upload the folder to a new GitHub repository, or import it directly into Vercel.
3. In Vercel, choose **Add New → Project**.
4. Select this project.
5. Framework preset: **Vite**.
6. Build command: `npm run build`.
7. Output directory: `dist`.
8. Click **Deploy**.

## Custom domain

After deployment:
1. Open the project in Vercel.
2. Go to **Settings → Domains**.
3. Add your domain.
4. Follow Vercel’s DNS instructions from your domain registrar.
